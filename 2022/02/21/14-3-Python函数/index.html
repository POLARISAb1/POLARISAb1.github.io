函数的作用

  • 函数是组织好的，可重复使用的，用来实现单一或相关联功能的代码段
  • 函数能提高应用的模块性和代码的重复利用率
  • python 内置函数：https://docs.python.org/zh-cn/3.8/library/functions.html

函数定义

  • def：函数定义关键词
  • function_name：函数名称
  • ()：参数列表放置的位置，可以为空
  • parameter_list：可选，指定向函数中传递的参数
  • comments：可选，为函数指定注释"""XXX"""
  • function_body：可选，指定函数体
